# Laptop Request Catalog Item – ServiceNow Project

##  Project Overview
This project implements a dynamic Service Catalog item in ServiceNow to automate the laptop request process.

##  Features Implemented
- Local Update Set creation
- Service Catalog item configuration
- Variable creation with correct order
- Catalog UI Policy implementation
- Client-side UI Action (Reset Form)
- Update Set export and import
- Functional testing
